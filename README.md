# Лабораторная работа №3: Конвертация карты глубины

## Описание
Консольное приложение для преобразования бинарных карт глубины в трехмерные модели с экспортом в формат OBJ и визуализацией.

##  Технические характеристики
- **Язык:** C++
- **Входной формат:** Бинарные карты глубины (.dat)
- **Выходной формат:** Wavefront OBJ (.obj)
- **Визуализация:** OpenGL с FreeGLUT
- **Платформа:** Windows

##  Быстрый старт

### Компиляция
```cmd
cl *.cpp /Fe:program.exe /I"D:\glut\freeglut\include" /EHsc /link "D:\glut\freeglut\lib\freeglut.lib" opengl32.lib glu32.lib gdi32.lib
```
### Запуск
##  Параметры командной строки

| Параметр | Описание | Пример использования |
|----------|----------|---------------------|
| `-i <файл>` | Указать входной файл карты глубины | `program.exe -i depthmap.dat` |
| `-o <файл>` | Указать выходной OBJ файл | `program.exe -o model.obj` |
| `-s <число>` | Задать масштаб модели | `program.exe -s 2.0` |
| `-v` | Включить 3D визуализацию | `program.exe -v` |
| `-h` | Показать справку | `program.exe -h` |
```cmd
program.exe -i DepthMap_10.dat -v
```
##  Управление в 3D-режиме
| Клавиша | Действие |
|---------|----------|
| `Стрелки ←/→` | Вращение по оси Y |
| `Стрелки ↑/↓` | Вращение по оси X |
| `W/S` | Вращение по оси X (альтернатива) |
| `A/D` | Вращение по оси Y (альтернатива) |
| `R` | Сброс камеры |
| `ESC` | Выход |
